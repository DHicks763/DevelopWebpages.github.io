Learning path

Start with HTML basics, move to CSS layout, then components and accessibility. Below are bite-sized examples you can copy.

  • HTML fundamentals: semantic structure, headings, lists
  • CSS basics: box model, flexbox, grid
  • Components: navbars, cards, forms
  • Accessibility & Performance: semantic landmarks, image alt, minification

Accessibility checklist

  • All images have descriptive alt.
  • Page landmarks (header, nav, main, footer) are present.
  • Color contrast meets 4.5:1 for text (AA) or 3:1 for large text.
  • Keyboard navigation is supported (tab order, :focus styles).

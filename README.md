This repo contains copy‑and‑paste, production‑ready website snippets and starter templates (HTML5, responsive meta, minimal CSS reset, accessible patterns, and basic JS) you can use to build static websites and deploy to GitHub Pages.
Semantic HTML Layout (Beginner‑Friendly Structure)
Shows how to structure a page using modern semantic tags.
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Semantic Layout Example</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>

  <header>
    <h1>My Webpage</h1>
    <nav>
      <a href="#">Home</a>
      <a href="#">Docs</a>
      <a href="#">Contact</a>
    </nav>
  </header>

  <main>
    <section>
      <h2>Introduction</h2>
      <p>This section introduces the topic.</p>
    </section>

    <article>
      <h2>Article Title</h2>
      <p>Content of the article goes here.</p>
    </article>
  </main>

  <footer>
    <p>© 2026 My Website</p>
  </footer>

</body>
</html>
